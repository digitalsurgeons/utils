# DS Utils

Digital Surgeons utility modules

**Installation**

	$ npm install ds-utils
	
**Example Usage**

	var Utils = {
		map: require('ds-utils/map'),
		hexMix: require('ds-utils/haxMix')
	};
	
**Module List**

- `map()` - map a value from one range to another
- `hexMix()` - return hex value at % point between two colors
